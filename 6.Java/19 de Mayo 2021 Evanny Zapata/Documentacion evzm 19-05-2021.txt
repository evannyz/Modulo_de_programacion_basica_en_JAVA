Se adjuntan dos ejercicios de tarea realizados con switch and case, 
el primero es la reproduccion de un ejercicio anterior realizado en pseudocodigo, 
se reestructura y se compila correctamente sin mayor problema, para el segundo ejercicio se simula una maquina de café, 
que me genera problemas inicialmente ya que no logro estructurar el programa de manera que capture las opciones mal digitadas
se restructura 3 veces el problema probando distintas combinaciones con switch e if else hasta que logro el objetivo,
para las clases continuamos con ejercicios en Java se continua ejercitando con switch and case y 
posteriormente se introducen los ciclos para-(mientras y hacer-mientras)
se investiga sobre formato de impresion de Strings en Java y el formato de los nuevos ciclos
